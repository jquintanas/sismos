Acerca de

SIS Geo Sismos Ecuador es un aplicativo multiplataforma con una versión web que emplea un paradigma orientación a objetos y su respectiva versión móvil en la cual se aplicó el paradigma orientado a aspectos y para la obtención de datos se usaron técnicas de scraping web para formatos XML desarrollado en Python usando un paradigma estructurado, ambas versiones fueron elaboradas en las medidas estándares de diseño y con el fin de poder orientar a los usuarios sobre los últimos sismos ocurridos en Ecuador y poder separarlos por regiones.
